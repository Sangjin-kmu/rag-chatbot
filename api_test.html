KDD Backend API Tester

인증 (Auth)
POST /auth/google Google OAuth 로그인
GET /auth/dev-token 개발용 JWT 발급

클릭하면 관리자 JWT가 자동 설정됩니다

GET /auth/me 현재 사용자 정보

JWT 토큰 필요 (위에서 설정)

문서 관리 (Documents)
POST /upload PDF 업로드 (관리자)
GET /documents 문서 목록 조회
DELETE /documents/{filename} 문서 삭제 (관리자)
POST /reset 전체 인덱스 초기화 (관리자)

주의: 모든 인덱스가 삭제됩니다

채팅 (Chat - RAG)
POST / RAG 질의응답
공지사항 크롤링
POST /crawl/notices 소융대 공지 크롤링 + PDF 변환

공지 본문을 PDF로 변환하고 첨부 PDF를 병합합니다. Gemini가 유효성을 판단하여 마감 지난 공지는 제외합니다. (관리자 JWT 필요)
