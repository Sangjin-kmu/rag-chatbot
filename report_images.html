Method Endpoint 설명 인증 Request Body Response Body

POST /reset 전체 인덱스 초기화 관리자 - { message }
GET /health 서버 헬스체크 불필요 - { status: "ok" }

Spring Security 설정
- CSRF 비활성화 (REST API)
- 세션 정책: STATELESS
- SecurityFilterChain 빈 등록
- 모든 경로 permitAll (JWT 개별 검증)
JWT 토큰 구조
- 알고리즘: HMAC-SHA256
- Payload: { email, name, iat, exp }
- 만료: 7일 (604,800초)
- 시크릿 키: 최소 32바이트
관리자 권한 분리
- 일반 사용자: 조회 API만 접근
- 관리자: 업로드/삭제/초기화 가능
- DOC_ADMIN_EMAILS 환경변수로 판별
- JWT 이메일 추출 후 관리자 목록 대조
